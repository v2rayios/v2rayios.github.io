小纸船的科学上网原理

小纸船之所以能够帮助用户突破网络限制，其核心在于代理技术的运用。简单来说，代理服务器就像是一个中转站，当用户访问受限网站时，请求会先发送到代理服务器，再由代理服务器代为访问目标网站，最后将获取的内容返回给用户。这种方式不仅能够隐藏用户的真实IP地址，还能绕过地理限制。

小纸船主要支持三种代理协议：

  1. HTTP代理：这是最基本的代理类型，适用于网页浏览等常规操作。
  2. SOCKS5代理：相比HTTP代理，SOCKS5支持更多类型的网络流量，包括电子邮件、文件传输等。
  3. VPN：虚拟专用网络，提供端到端的加密连接，安全性更高。

FAQ

PassWall2 策略组如何自动优化网络访问？
PassWall2 结合策略组和延迟检测功能，自动选择最佳节点处理网页、视频和游戏流量。无需手动切换，保证访问速度和稳定性，提升整体网络体验。
Netch 的规则代理与全局代理如何选择？
全局代理会将所有流量通过节点转发，而规则代理只针对指定 IP、端口或应用。用户可根据需求选择，以优化带宽利用和流量分流，确保不同应用的最佳体验。
Mellow 是否支持 IPv6 流量代理？
Mellow 支持 IPv6，只需在配置文件中启用 IPv6 DNS 解析并确认操作系统开启 IPv6 路由功能。部分节点若不支持 IPv6，则相关请求会自动回退到 IPv4，保证兼容性。
V2Ray 如何实现分流规则？
V2Ray 使用 routing 模块实现分流，可根据目标域名、IP、端口或协议类型决定流量走向。用户可自定义多个 outbound，并通过规则实现国内直连、国外代理等智能路由功能。
Quantumult X 支持 VLESS 吗？
Quantumult X 目前官方未原生支持 VLESS，但通过手动修改配置或第三方脚本可实现兼容。若主要使用 VLESS 协议，可考虑使用 Sing-box 或 Shadowrocket 作为替代。
如何防止 Clash 被系统防火墙阻断？
请在防火墙设置中允许 Clash 的执行文件通过网络访问，并开放其监听端口。若使用 Windows Defender，可在防火墙高级规则中手动添加入站和出站规则，确保代理通信正常。
Trojan-Go 的多路复用（mux）如何开启？
Trojan-Go 在配置文件中通过启用 mux 参数即可实现多路复用。此功能允许多个连接共享同一隧道，减少握手延迟并提升整体传输效率，尤其在移动网络中表现出色。
Trojan 如何实现伪装？
Trojan 使用 TLS 协议模拟正常的 HTTPS 流量，防火墙难以区分其与普通网站访问的区别。通过伪装域名、SNI 和证书匹配机制，它能在保障安全的同时隐藏真实代理流量。
如何把 Clash 规则与本地 hosts 文件结合使用？
可在 Clash 的 rules 中优先设置域名直连规则，并在系统 hosts 中添加常用站点的固定解析以加速 DNS 解析。注意 hosts 优先级高于 DNS，因此修改 hosts 需谨慎以避免与真实解析冲突。
Mihomo 是什么？
Mihomo 是 Clash Premium 的开源替代项目，兼容原版 Clash 配置文件并扩展了部分新功能。它性能优异、内存占用低，适用于希望自定义构建代理环境的高级用户。
